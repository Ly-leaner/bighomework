大作业第一阶段报告
1、基本设计思路和流程：
基础功能部分，通过窗口类”MainWindow”的构造函数实现”TowerPosition”, “Tower”, 
“enemy”, “bullet”等类的调用，同时通过Qtimer以及重载paintEvent和mousePressEvent实现对
窗口的基本控制。升级功能部分，怪物的升级主要通过对 “enemy”类的继承和派生实现，在防御方的
分值方面，首先设置每次安置防御塔所需的分值和每次消灭怪物获得的分值，并在此基础上对从”Tower”
派生而来的升级防御塔进行管理。
2、已经实现的功能：
地图的绘制；防御塔的绘制、攻击范围；敌人的绘制、波数、血量、移动方向、移动速度、
基本的碰撞检测。其中，防御塔安置点和敌人的波数信息可以通过数组或.xml文件进行管理，
利用数组实现管理的方式在代码中以注释的方式给出，利用.xml文件管理则通过”PlistReader”类
从资源文件中读取”TowersPosition.plist”和”Waves.plist”两个文件实现，两个.plist文件是在网络
模板的基础上，通过编辑器Plist Editor编辑而成。子弹类完成了一部分，还需要调bug。
